# learncoro - 轻量级协程库学习性项目源码
一些用C语言实现轻量级协程库的学习性项目源码，旨在通过实践深入理解操作系统并发原理
## 📖 前言

本项目旨在通过用C语言从零开始实现协程库，深入学习操作系统中的并发机制，因对**Github**还不熟悉，因此只是公开源码，到时Rust重写考虑做成一个真正的项目，
开发过程中将逐步记录学习笔记，后期整理成完整的技术文档，供其他学习者参考。

## 📚 项目定位
**学术参考实现** - **仅提供源码**学习操作系统并发原理

**技术栈演进计划：**
1. 第一阶段：用C语言实现完整的协程库基础功能
2. 第二阶段：用Rust重写整个项目（作为独立项目）
3. 第三阶段：在Rust版本基础上进行功能扩展
## 🚧 项目状态

**当前阶段：** 早期开发阶段  
目前已完成基本框架设计和部分功能实现，仍存在一些已知问题需要修复。

## 📋 功能规划

### 核心功能
- [x] **上下文管理**：使用WINAPI实现上下文切换
- [ ] **上下文管理**：用纯C语言结构体模拟上下文结构并实现切换
- [ ] **调度器封装**：实现可扩展的调度器框架
- [ ] **调度算法**：实现多种协程调度策略
- [ ] **协程通信**: 实现协程通道通信
- [ ] **异步IO集成**: 实现异步IO
- [ ] **协程库验证**: 开发微型项目验证此协程库的功能

### 扩展功能
- [ ] **Python接口**：提供Python绑定
- [ ] **性能分析工具**：协程运行状态监控
- [ ] **调试支持**：协程调用栈追踪
- [ ] **多平台支持**：Linux/macOS跨平台适配
