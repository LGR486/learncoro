# learncoro - 从零实现协程库和其应用
用C语言从零开始实现协程库，并验证协程库的功能开发微型应用，独立开发，边学边做。
## 📖 前言

本项目旨在通过用C语言从零开始实现协程库，深入学习操作系统中的并发机制。开发过程会用到轻微AI，但是只是作为学习参考，设计方案是通过大量思考和参考设计的，**不存在大量复制AI的行为**。因对**Github**还不熟悉，因此暂时考虑只是公开源码，到时Rust重写考虑做成一个真正的项目，开发过程中将逐步记录学习笔记，后期整理成完整的技术文档，供其他学习者参考。

## 📚 项目定位
**学术参考实现** - **仅提供源码**学习操作系统并发原理，以及应用。

**项目参考** - 本项目开发将以操作系统原理作为指导书学习，同时参考腾讯libco源码，rust重写时将参考tokio源码。

**技术栈演进计划：**
1. 第一阶段：用C语言实现完整的协程库基础功能
2. 第二阶段：用Rust重写整个项目（作为独立项目）
